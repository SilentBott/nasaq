# 🌙 نَسَق | Nasq - رفيقك في ختم القرآن الكريم

تطبيق **"نَسَق"** هو منصة ويب متطورة تهدف لتنظيم ومتابعة قراءات القرآن الكريم بأسلوب عصري ودقيق. تم تصميم التطبيق ليتجاوز مجرد الحجز التقليدي، ليوفر تجربة "نَسقية" متكاملة تضمن المزامنة اللحظية والدقة البرمجية في حساب الورد.

---

## ✨ المميزات الرئيسية (Key Features)

- **نظام المجموعات الذكي**: أنشئ ختمتك الشخصية أو انضم لمجموعات تشاركية (مسجد، عائلة، أصدقاء) بضغطة زر.
- **المزامنة اللحظية (Real-time)**: تحديثات فورية لجميع المستخدمين عند حجز أو ختم أي سورة بفضل تقنية **Supabase Realtime**.
- **منطق الحساب الدقيق (Unique Verse Logic)**: حساب نسبة الإنجاز بناءً على أرقام الآيات "الفريدة" المقروءة؛ مما يمنع تكرار الحساب في حالات التداخل ويضمن دقة التقدم.
- **التحكم النسبي في الخط (Relative UI Scaling)**: ميزة حصرية تتيح للمستخدم تكبير وتصغير حجم الخط في الموقع بالكامل (الأسماء، الآيات، الواجهات) بنسق متناسق ومريح للعين.
- **واجهة حجز مطورة**: خيارات ذكية لاختيار نطاق القراءة تبدأ بـ "أوّل آية" وتنتهي بـ "آخر السورة" لتسريع عملية التسجيل.
- **الختم السريع**: ميزة "الضغط المطول" على كارت السورة لختمها بالكامل فوراً في حالات القراءة خارج التطبيق.
- **نظام الفلترة المتقدم**: عرض مخصص للسور (الكل، المكتملة، المتبقية، ما نحن فيه حالياً).

## 🛠️ التقنيات المستخدمة (Tech Stack)

| التقنية          | الوصف                                                |
| :--------------- | :--------------------------------------------------- |
| **React (Vite)** | بناء واجهة المستخدم وضمان سرعة الاستجابة الفائقة     |
| **Supabase**     | قاعدة البيانات وإدارة المزامنة اللحظية والـ Auth     |
| **Tailwind CSS** | التنسيق العصري "Dark Mode" والمتجاوب مع جميع الشاشات |
| **Lucide React** | مكتبة الأيقونات التفاعلية البسيطة والأنيقة           |
