# 🌙 نَسَق | Nasq - رفيقك الذكي في ختم القرآن الكريم

تطبيق **"نَسَق"** ليس مجرد منصة لتسجيل الورد اليومي، بل هو **بيئة قرآنية ذكية** مبنية بأحدث تقنيات الويب. تم هندسة التطبيق ليعالج أعقد المشاكل البرمجية المرتبطة بالقرآن الكريم (كالرسم العثماني، وحساب تداخل الآيات، والمزامنة اللحظية)، ليقدم تجربة مستخدم (UX) فائقة النعومة والدقة.

---

## ✨ المميزات الحصرية (Advanced Features)

- 👥 **ختمات مخصصة ودخول فوري:** لا حاجة لإنشاء حسابات معقدة؛ ادخل فوراً وابدأ ختمتك الشخصية، أو شارك الأجر وانضم لمجموعة عامة بضغطة زر.
- 🔥 **إحصائيات دقيقة وسلسلة الالتزام:** نظام منطقي معقد يعتمد على `PostgreSQL Triggers` و `generate_series` لحساب الآيات المقروءة فعلياً دون تكرار إذا تقاطعت قراءات الأعضاء (Zero Overlap)، بالإضافة لنظام (الشعلة) لتحفيز القراءة اليومية.
- 📡 **رادار الختمات والمزامنة اللحظية:** ميزة (Gamification) ذكية ترصد المجموعات التي اقتربت من الختم (98% فأكثر) لتحفيز المشاركة، مع تحديثات حية لنسب الإنجاز بفضل تقنية Supabase Realtime.
- 🔍 **محرك بحث بالرسم العثماني:** نظام `Regex` خارق الذكاء يبحث في القرآن الكريم متجاهلاً التشكيل، همزات الوصل، والألف الخنجرية، ليقدم نتائج دقيقة ومظللة بامتياز.
- 📴 **وضع عدم الاتصال (Offline-First):** اقرأ وسجل وردك بدون إنترنت! يقوم النظام بحفظ قراءاتك في الـ `localStorage` ومزامنتها تلقائياً بالخلفية فور عودة الاتصال.

---

## 📐 تجربة مستخدم متكيفة (Adaptive UX)

- **مقياس الخط الديناميكي (Container Queries):** استخدام تقنية `cqi` في CSS لضمان تكبير وتصغير المصحف والواجهات بنسق رياضي مريح للعين متزامن مع إعدادات المستخدم.
- **واجهة حجز مطورة:** خيارات ذكية لاختيار نطاق القراءة تبدأ بـ "أوّل آية" وتنتهي بـ "آخر السورة" لتسريع عملية التسجيل.

## 🛠️ التقنيات المستخدمة (Tech Stack)

| التقنية                   | الوصف                                                       |
| :------------------------ | :---------------------------------------------------------- |
| **React (Vite)**          | بناء واجهة مستخدم سريعة الاستجابة (SPA)                     |
| **Supabase (PostgreSQL)** | قاعدة البيانات، الـ Triggers المعقدة، والمزامنة اللحظية     |
| **Tailwind CSS**          | تنسيق عصري، دعم الـ Dark Mode، وتصميم متجاوب بالكامل        |
| **Vite PWA**              | تحويل الموقع لتطبيق هاتف قابل للتثبيت (Progressive Web App) |
| **Lucide React**          | أيقونات تفاعلية بسيطة وأنيقة                                |

---

## 🚀 رحلة المستخدم (User Flow)

1. **دخول سريع:** اكتب اسمك وابدأ.
2. **اختيار المسار:** تابع وردك الخاص، أو انضم لمجموعة.
3. **تحديد الورد:** حدد الآيات والسور التي أتممتها بمرونة فائقة.
4. **تتبع الإنجاز:** حافظ على شعلتك مشتعلة وراقب رادار الختمات.
