# 🌙 ختمة - Khatmah Tracker

تطبيق **"ختمة"** هو منصة ويب متطورة تهدف لتسهيل تنظيم ومتابعة قراءات القرآن الكريم، سواء كان ذلك بشكل فردي خاص أو ضمن مجموعات تشاركية للتنافس في الخير.

---

## ✨ المميزات الرئيسية (Key Features)

- **نظام المجموعات**: إمكانية إنشاء ختمات عامة أو الانضمام لمجموعات موجودة لمشاركة التقدم مع الآخرين.
- **المزامنة اللحظية**: تحديثات فورية لجميع المستخدمين عند حجز أو ختم أي سورة بفضل تقنية Real-time من Supabase.
- **مؤشر الإنجاز (Progress Bar)**: شريط مرئي أسفل كل سورة يوضح نسبة ما تم قراءته بدقة بناءً على عدد الآيات.
- **الحجز الذكي والختم السريع**: واجهة سهلة لاختيار نطاق آيات معين، مع خاصية "الضغط المطول" لختم السورة بالكامل فوراً.
- **نظام الفلترة المتقدم**: عرض السور المتبقية، المكتملة، أو التي تتابعها أنت فقط لتسهيل التركيز.
- **واجهة مستخدم عصرية**: تصميم "Dark Mode" مريح للعين مبني باستخدام Tailwind CSS لضمان أفضل تجربة قراءة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

| التقنية          | الوصف                                    |
| :--------------- | :--------------------------------------- |
| **React (Vite)** | بناء واجهة المستخدم وضمان سرعة التصفح    |
| **Supabase**     | قاعدة البيانات وإدارة المزامنة الفورية   |
| **Tailwind CSS** | التنسيق العصري والمتجاوب مع جميع الشاشات |
| **Lucide React** | مكتبة الأيقونات التفاعلية داخل التطبيق   |
